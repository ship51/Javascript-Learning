How to add the extension to browser?

-> GO to browser
-> Go to extensions page (chrome://extensions)
-> Enable developer mode
-> Click 'Load Unpacked'
-> Select folder in which the extension is saved 
(make sure to add a relevant icon which makes it easy to distinguish)



How to edit code?

-> Check manifest .json
-> content.js changes selective sites (changes background of p elements)
-> content2.js changes selective sites (changes background of p elements)
-> background.js is for the background of the browser outside of the web page; 
console can be accessed by clicking on "background page" from extensions page in browser
-> Reload extension every time after editing the code
